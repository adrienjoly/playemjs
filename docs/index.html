Loader

This class provides helpers to load JavaScript resources and JSON data.

new Loader()
Instance Members
▸ loadJSON(src, cb)

Loads and returns a JSON resource asynchronously, using XMLHttpRequest (AJAX).

loadJSON(src: string, cb: dataCallback)
Parameters
src (string) HTTP(S) URL of the JSON resource to load.
cb (dataCallback) Callback function with request's data as first parameter.
▸ includeJS(src, cb)

Loads a JavaScript resource into the page.

includeJS(src: string, cb: errorCallback)
Parameters
src (string) HTTP(S) URL of the JavaScript resource to load into the page.
cb (errorCallback) Callback function with error as first parameter, if any.
▸ loadJSONP(src, cb)

Loads and returns a JSON resource asynchronously, by including it into the page (not AJAX).

loadJSONP(src: string, cb: function)
Parameters
src (string) HTTP(S) URL of the JSON resource to load.
cb (function) Callback function, called by the resource's script.

Playem

Plays a sequence of streaming audio/video tracks by embedding the corresponding players into the page.

Events:

  • "onError", {code,source}
  • "onReady"
  • "onPlay"
  • "onPause"
  • "onEnd"
  • "onTrackInfo", track{}
  • "onTrackChange", track{}
  • "loadMore"
new Playem(playemPrefs: Object)
Parameters
playemPrefs (Object) Settings and preferences.
Name Description
playemPrefs.loop Boolean If true, the playlist will be played infinitely. (default: true)
playemPrefs.playTimeoutMs Number Number of milliseconds after which an error event will be fired, if a tracks was not able to play. (default: 10000, i.e. 10 seconds)
